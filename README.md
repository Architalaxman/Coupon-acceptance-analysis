# Coupon Acceptance Analysis

## 📊 Project Overview
This project focuses on analyzing customer behavior to understand coupon acceptance patterns. The study explores how demographic factors (age, gender, income, marital status) and contextual factors (weather, time, destination, and driving conditions) influence decision-making.

## 🎯 Objective
To identify key trends and customer segments that are more likely to accept coupons, helping businesses design more effective and targeted marketing strategies.

## 🛠 Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📁 Dataset Information
- Total Records: 12,684
- Total Features: 25
- Includes demographic, behavioral, and contextual attributes
- Target Variable: Coupon Acceptance (Yes/No)

## 📈 Key Insights
- Young adults (21–31) are more likely to accept coupons
- Higher acceptance in sunny weather and warmer temperatures
- Customers with frequent visits to coffee houses and restaurants show higher acceptance
- Shorter travel distance increases acceptance likelihood
- 1-day coupons perform better than short-duration (2-hour) offers

## 💡 Business Recommendations
- Use **targeted marketing** based on customer behavior.
- Apply **differential discounting**:
  - Lower discounts for high-acceptance groups
  - Higher discounts for low-acceptance segments
- Focus on personalized and time-flexible offers

## 📌 Conclusion
Coupon acceptance is predictable using data analysis. Businesses can improve promotional effectiveness and maximize profit by leveraging customer insights and behavior patterns.
